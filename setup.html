Setup guide

Cloud Function + Tasker → auto-capture ICICI SMS

1 Deploy the Cloud Function

Open a terminal in your project folder and run these commands. Make sure you have the Firebase CLI installed.

Install Firebase CLI (if not installed)
npm install -g firebase-tools
Login + deploy
cd "D:\Projects\Claude Projects\Personal Finance Tracker"
firebase login
cd functions
npm install
cd ..
firebase deploy --only functions
After deploy: Firebase will print your function URL. It'll look like:
https://parsesms-XXXXX-uc.a.run.app (asia-south1 region)

Copy that URL — you'll need it for Tasker in Step 3.
Your API key (set in functions/index.js)
myfinance_viyas_2026
Want a different API key? Open functions/index.js, change the API_KEY value on line 8, then redeploy.

3 Set up Tasker

Install Tasker from Play Store (₹350, one-time). Grant SMS permission when prompted.
Profile: SMS trigger
  1. Open Tasker → Profiles tab → + → Event
  2. Select Phone → Received Text
  3. Set Type: SMS
  4. Set Sender: ICICI
  5. Leave Content empty
  6. Hit back ← to save
Task: Send SMS to Cloud Function
  1. Tasker prompts you to create a task → name it SendToFirebase
  2. Add action → Net → HTTP Request
  3. Set the fields:
Method:  POST

URL:     YOUR_FUNCTION_URL_HERE
         (paste the URL from Step 1)

Headers: Content-Type: application/json
         x-api-key: myfinance_viyas_2026

Body:    {"sms": "%evtprm1"}
What's %evtprm1? That's Tasker's variable for the SMS body text. It auto-fills with the actual SMS content when triggered.
Optional: Add a notification on success
  1. Add action → Alert → Flash
  2. Set Text: Transaction saved!

This shows a brief toast on your phone whenever a transaction is auto-captured.

Final checklist
  • Hit back ← until you see the profile listed
  • Make sure the profile toggle is ON (green)
  • Tap the checkmark ✓ to save

4 Verify end-to-end

Use the test button in Step 2 to send a sample transaction, then open My Finance to confirm it shows up.

The flow once everything is set up:
Bank SMS arrives → Tasker intercepts → POSTs to Cloud Function → Function parses + writes to Firestore → My Finance shows it in uncategorized queue → You tap to categorize
