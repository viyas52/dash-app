What gets tracked
✓ Read
• Bank SMS with a known DLT sender ID (HDFCBK, ICICIB, SBIINB…)
• Body must contain transaction keywords (debited / credited / UPI / Rs / ₹…)
• Only your linked accounts — anything else is dropped server-side
✗ Never read
• WhatsApp, Instagram, Gmail, social, news, any non-messaging app
• OTPs, promotional SMS, personal messages
• Any SMS whose sender isn't a known bank header
• Bank SMS for accounts you haven't linked
Diagnostics only stores bank txn SMS we actually forwarded — nothing else. Code is open at github.com/viyas52/dash-app.
Smart format learning
Learn new bank formats
If your bank isn't recognised, the SMS is sent once to an AI to learn its format. After that it's parsed on-device — no further AI use.
Off → only built-in bank formats work; an unrecognised bank shows "format not recognized" until support is added manually. Your existing transactions are never sent.

Welcome, there
My Finance auto-tracks your money by reading bank SMS — no manual entry. Let's get you set up in 3 quick steps.
🏦
Link your bank account
So we know which account to track and can show your live balance. Only linked accounts get tracked — everything else is ignored.
📱
Install Finance Companion
A tiny Android app that forwards bank SMS so your transactions appear here automatically. Open-source, only reads bank SMS.
⬇ Download APK
Step by step
1
Open the APK from your downloads
Chrome will warn "unknown source" — tap Settings → toggle "Allow from this source" → back → Install
2
If Play Protect blocks it
You'll see "App blocked to protect your device". Tap More details → Install anyway. The app is open-source — this warning is generic for any sideloaded app.
3
Allow restricted settings (Android 13+)
Open Android Settings → Apps → Finance Companion → tap the ⋮ (3 dots top-right) → "Allow restricted settings". Without this, the notification toggle below stays greyed out.
4
Enable notification access
Android Settings → Apps → Special access → Notification access → find Finance Companion → toggle ON → confirm "Allow". This is what lets the app see your bank SMS notifications.
5
Come back here, refresh the page
Your username + API key auto-sync to the companion app. A green "📱 Enable SMS forwarding" banner will appear once — tap it for one last permission check, then you're done.
🎉
You're all set
Try it out:
1. Make a small UPI payment (₹1 to yourself)
2. Wait for the bank SMS (~10 sec)
3. Tap the notification to categorize

That's it — every future transaction works the same way.
